This is a C project for *STM32F103C8T6*

Core features:
1. Show time
2. Show weather (pressure, temperature, humidity)
3. Timer with alarm (sound and light)

Pins:
1. Screen (GME12864):
- SCL - B6
- SDA - B7

2. Weather (BME280):
- SCL - B10
- SDA - B11

3. Encoder (KY-040):
- CLK - A15
- DT - B3
- SM - not used

4. Alarm (LED + Buzzer):
- A1 

5. Buttons:
- First - A3
- Second - B1
